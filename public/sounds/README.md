Drop replacement sound effects in this folder with these exact names:

- `card-hover.wav`: when a player hovers a hand card
- `card-pick.wav`: when a player selects a card
- `chat-send.wav`: when a chat message is sent
- `game-win.wav`: when the local player wins the game

The app uses generated Web Audio beeps as a fallback if a file is missing or cannot play.
